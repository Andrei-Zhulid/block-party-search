# Run Instructions

## Backend solution

1. Open api folder:

```
cd api/
```

2. Install dependencies:

```
npm install
```

3. Run:

```
npm run dev
```

## Frontend solution

**Pre-requirements:** Make sure backend service is already running.

1. Open frontend folder:

```
cd frontend/
```

2. Install dependencies:

```
npm install
```

3. Run:

```
npm run serve
```

4. Open http://localhost:8080/
